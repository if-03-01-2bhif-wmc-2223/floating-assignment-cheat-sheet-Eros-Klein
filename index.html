Webservers

An URL(Uniform Resource Locator) is a way too get resources of the web. The general components of an URL are: scheme, subdomain, top level Domain, second level Domain, subdirectory
A web server understands URLs(Uniform Resource Locators) and HTTP(Hypertext Transfer Protocol), and connects to the internet and supports physik data interchange, with other devices connected to the web
It saves textual, aural, visual data with either files on the server or some kind of database
A web client(Browser) uses the web requests to provide access to resources
The head element contains metadata and you can use em or mark tag to make your text look different
